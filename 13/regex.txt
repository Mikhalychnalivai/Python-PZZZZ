# 1. Найдите все натуральные числа (возможно, окружённые буквами) - g
# \d+

# 2. Найдите все «слова», написанные капсом (то есть строго заглавными), возможно внутри настоящих слов (аааБББввв) - g
# [A-ZА-ЯЁ]{2,}

# 3. Найдите слова, в которых есть русская буква, а за ней цифра - g
# [а-яА-ЯёЁ]\d

# 4. Найдите все слова, начинающиеся с русской или латинской большой буквы (\b — граница слова) - g
# \b[A-ZА-ЯЁ]\w+

# 5. Найдите слова, которые начинаются на гласную (\b — граница слова) - g
# \b[аеёиоуыэюяАЕЁИОУЫЭЮЯaeiouAEIOU]\w+

# 6. Найдите все натуральные числа, не находящиеся на границе слова - g
# \B[0-9]+\B

# 7. Найдите строчки, в которых есть символ * (. — это точно не конец строки!) - gm
# ^.*\*.*$

# 8. Найдите строчки, в которых есть открывающая и когда-нибудь потом закрывающая скобки - gm
# ^.*\(.*\).*$

# 9. Выделите одним махом весь кусок оглавления (в конце примера, вместе с тегами) - g
# <a href=\"#\d+\">[\s\S]*</a>;

# 10. Выделите одним махом только текстовую часть оглавления, без тегов - g
# (?<=>)[^<]+(?=<\/a>) 

# 11. Найдите пустые строчки - gm
# ^\s*$

12. Найти все теги, не включая их содержимое - gm
<[^>]+>
